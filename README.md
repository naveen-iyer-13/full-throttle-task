# Full Throttle task

### Installing and Running

```
- git clone git@github.com:naveen-iyer-13/full-throttle-task.git
```
```
- npm install
```
```
- npm start
```


## Built With

* [React](https://reactjs.org/) - The JavaScript Library
* [Ant Design](https://ant.design/) - A design library
* [FullCalendar](https://fullcalendar.io/) - A JavaScript Calendar library
